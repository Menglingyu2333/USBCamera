@par Example Description 

  This demo provides a basic communication between USART2 and USART3 in 
  Half-Duplex mode using flags.If received data right ,the Three leds will 
  turn on.
  
  Set-up 
  - Connect USART2 Tx pin (PD.05) to USART3 Tx pin (PC.10) and connect a 
  pull-up resistor to this line (10K).

@par Example Description

This example demonstrates and explains how to use the RTC peripheral. As an
application example, it demonstrates how to setup the RTC Calendar function. 
The LSE (32768Hz) clock is used as RTC clock source and prescaler is 32768. 
Using the USART1 print the data.

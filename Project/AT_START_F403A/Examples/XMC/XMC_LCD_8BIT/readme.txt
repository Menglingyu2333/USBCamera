  @brief
    This example provides a basic example of how to use the XMC firmware library and
	an associate driver to perform read/write operations on the LCD. The LCD drive IC
	is ST7789V and  will display some number, it's actually showing pictures.This demo 
	support 100PIN and 64PIN pakeage.
	
	The pin to pin with LCD and XMC:
	GPIO      PB10    LCD_RESET
	XMC_A0    PC3     LCD_RS
	XMC_NE4   PC4     LCD_CS
	XMC_NWE   PD2     LCD_WR 
	XMC_NOE   PC5     LCD_RD 
	XMC_D0	  PB14	  DATA[0]
	XMC_D1	  PC6	  DATA[1]
	XMC_D2	  PC11	  DATA[2]
	XMC_D3	  PC12	  DATA[3]
	XMC_D4	  PA2	  DATA[4]
	XMC_D5	  PA3	  DATA[5]
	XMC_D6	  PA4	  DATA[6]
	XMC_D7	  PA5	  DATA[7]

@par Example Description 
  This demo  provides a description of how to use DMA with TMR1 Update request
  to transfer Data from memory to TMR1 Capture Compare Register3.
  
  Set-up 
    - Connect the following pins to an oscilloscope to monitor the different 
      waveforms:  
        - TMR1 CH3  (PA.10)
        - TMR1 CH3N (PB.15)

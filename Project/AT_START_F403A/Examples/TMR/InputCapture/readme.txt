@par Example Description 
  This demo shows how to use the TIM peripheral to measure the frequency of an 
  external signal.
  the minimum frequency value to measure is 3.66 KHz(240M/65536). 
  Set-up 
    - Connect the external signal to measure to the TIM3 CH2 pin (PA.07).

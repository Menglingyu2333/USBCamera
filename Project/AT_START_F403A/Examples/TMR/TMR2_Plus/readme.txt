@par Example Description 
    This demo is based on the AT-START-F403A board,in this demo,PA0 ,PA1 ,PA2 and PA3 output pwm waveform.    
    In this case , CH1 CH2 CH3 CH4 frequency = TMRclock / (Period+1) / (DIV+1) = 183 HZ  Systemclock = 192Mhz
    CH1 dutycycle = 50%
    CH2 dutycycle = 25%
    CH3 dutycycle = 12.5%
    CH4 dutycycle = 6.25%

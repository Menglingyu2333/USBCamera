@par Example Description 

  This demo is based on the AT-START-F403/AT-START-F413/AT-START-F415/AT-START-F403A/AT-START-F407 board ,which can be select on toolbar"Select Target".
  In this demo,show how to get random nuber by RTC and UID ,and USART1 printf the result.
  Through config #define EnableRTCAsSeed value 0 or 1,to opnen or close RTC as seed for random.
